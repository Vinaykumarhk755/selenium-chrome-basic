Java Selenium Project (No Maven) with proper package folder structure

Steps:
1. Import this as a Java project in your IDE.
2. Make sure 'src' is marked as source folder.
3. Replace the jars in 'lib' with actual Selenium jars downloaded from https://www.selenium.dev/downloads/
4. Download chromedriver from https://chromedriver.chromium.org/downloads matching your Chrome version and place it inside 'lib' folder.
5. Run the main class com.example.OpenGoogleChrome
